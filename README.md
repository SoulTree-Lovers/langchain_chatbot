# AI 기술 전문가 챗봇

이 프로젝트는 PDF 문서를 기반으로 한 AI 기술 전문가 챗봇입니다.

## 주요 기능
- PDF 문서 업로드 및 분석
- 문서 기반 질의응답
- 문서 요약 기능

## 설치 방법
```bash
pip install -r requirements.txt
```

## 실행 방법
```bash
streamlit run practice.py
```

## 환경 변수 설정
`.env` 파일을 생성하고 다음 환경 변수를 설정하세요:
```
OPENAI_API_KEY=your_openai_api_key_here
```
